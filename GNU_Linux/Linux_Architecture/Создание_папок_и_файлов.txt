Создание папок и файлов в Linux
1. Создать несколько папок в текущей директории:
	mkdir Folder1 Folder2 FolderN		#mkdir названия папок через пробел
2. Создать многоуровневую директорию в текущей:
	mkdir -p a/b/c/d
	#a, b, c, d - имена директорий
3. Создание файлов с использованием перенаправления
	3.1. Создать текстовый файл и записать в него список файлов и папок
		ls>tmp
	3.2. Создать простой текстовый файл tmp и записать в него 'privet'
		echo 'privet'>tmp
	3.3. Создать пустой текстовый файл tmp
		>tmp
4. Создание файлов с использованием утилиты cp
		cp /dev/null name_file 
		Команда cp создана для копирования файлов.
		Если в качестве файла указать dev/null, то будет создан новый файл
5. Создать файл используя функцию перенаправления командного процессора и команду cat
	5.1. Синтаксис команды для набора из терминала:
		cat > name_file			### Набираем текст, нажимаем комбинацию CTRL + D -> Готово
	5.2. Создание текстового файла с включенным в него текстом:
		cat > name_file  << EOF 
		TEXT_1
		TEST_2
		TEXT_N
		EOF
6. Создать файл, используя текстовый редактор vim
	vim -c wq name_file 
7. Создание файла с помощью сенсорной команды
	Команда touch позволяет нам обновлять метки времени в существующих файлах и каталогах, а также создавать новые пустые файлы.
	touch file1.txt
	touch file1.txt file2.txt fileN.txt
	Если файл с указанным именем не существует, приведенная выше команда создаст его, в противном случае она изменит свои временные метки.
8. Создание файла с использованием Heredoc

	Здесь документ или Heredoc — это тип перенаправления, который позволяет передавать команде несколько строк ввода.
	Этот метод в основном используется, когда вы хотите создать файл, содержащий несколько строк текста, из сценария оболочки.
	Например, чтобы создать новый файл file1.txt вы должны использовать следующий код:
		cat << EOF > file1.txt
		Some line
		Some other line
		EOF
