The Linux distribution world consists of full-blown [полномасштабный] Linux distributions that include just about every application imaginable, as well as specialized Linux distributions that include applications focused only on a special function.

Although the bash shell program is automatically started at login, whether a shell command line interface (CLI) is presented depends on which login method is used.
(..., наличие интерфейса командной строки оболочки (CLI) зависит от используемого метода входа в систему.)
If a virtual console terminal is used to log in, the CLI prompt is automatically presented, and you can begin to type shell commands.
(Если для входа в систему используется терминал виртуальной консоли, автоматически отображается приглашение CLI, и вы можете начать вводить команды оболочки.)
However, if you log into the Linux system via a graphical desktop environment, you need to start a graphical terminal emulator to access the shell CLI prompt.
(Однако, если вы входите в систему Linux через графическую среду рабочего стола, вам необходимо запустить эмулятор графического терминала, чтобы получить доступ к приглашению оболочки CLI)
