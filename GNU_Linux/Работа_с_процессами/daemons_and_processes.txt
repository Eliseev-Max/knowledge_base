Понятие "демон" (daemon) относится к процессам, а не к файлам.
По этой причине нет смысла в поиске демонов в файловой системе.
Пояснение:
	программа - это исполняемый файл (виден в выводе ls);
	процесс - это (запущенный) экземпляр этой программы (виден в выводе ps).
Поиск демонов сводится к поиску процессов, которые запускаются без подключенного к ним управляющего терминала.
Пример поиска:
	$ ps -eo 'tty,pid,comm' | grep ^?
	# Поле вывода tty содержит "?", если процесс не имеет управляющего терминала.
	# ! Проблема: если система работает в графической среде (GUI):
	# Поскольку программы с графическим интерфейсом (например, Chromium) не привязаны к терминалу,
	# они также появляются в выводе.
	
	# В стандартной системе, где root не запускает графические программы, вы можете просто ограничить предыдущий список процессами root:
	$ ps -U0 -o 'tty,pid,comm' | grep ^?
	# ! Проблемы:
	# 1. Если root запускает графические программы, они будут отображаться.
    # 2. Демоны, запущенные без привилегий root, не будут отображаться.

‼ Демоны, которые запускаются во время загрузки, обычно запускаются от имени root.

(?) Как отобразить все программы без управляющего терминала, но не программы с графическим интерфейсом?
[→] Программа для вывода списка GUI-процессов: xlsclients
	Чтобы исключить все GUI-программы, нужно запустить xlsclients с параметром --deselect
	
[1] Сначала мы построим список всех программ GUI, для которых у нас есть запущенные процессы:
		$ xlsclients | cut -d' ' -f3 | paste - -s -d ','
	# Мы только что получили список команд
	 
[2] Теперь, ps имеет переключатель -C, который позволяет нам выбирать по имени команды.
	# введем список команд в командную строку ps:
		$ ps -C "$(xlsclients | cut -d' ' -f3 | paste - -s -d ',')" --deselect
	# Теперь у нас есть список всех не-GUI процессов.
	# Отфильтруем результат по признаку: "без TTY"
		$ ps -C "$(xlsclients | cut -d' ' -f3 | paste - -s -d ',')" --deselect -o tty,args | grep ^?
	# Последний grep отбирает все строки, начинающиеся с "?", то есть все процессы без управляющего tty.
[+] Улучшайзинг:
	1. Можно исключить потоки ядра, которые не являются процессами:
		$ ps -C "$(xlsclients | cut -d' ' -f3 | paste - -s -d ',')" --ppid 2 --pid 2 --deselect -o tty,args | grep ^?
	2. Добавим несколько столбцов информации:
		$ ps -C "$(xlsclients | cut -d' ' -f3 | paste - -s -d ',')" --ppid 2 --pid 2 --deselect -o tty,uid,pid,ppid,args | grep ^?
