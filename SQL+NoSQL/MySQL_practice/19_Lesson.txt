Задавать ограничения (критерии) вывода в SELECT можно с помощью WHERE:
...WHERE col_1 = ...(<>, >,>=,<,<=)
...WHERE col_2 like 'M%' -- Выведет все записи, для которых значения в столбце col_2
начинаются с 'M'.
HAVING
Оператор HAVING используется, когда значения сгруппированы с помощью GROUP BY или агрегирующих
функций (count(), sum(), min(), max() )

Производительность БД. Индекс.
Если БД ещё недостаточно заполнена данными, проводить оптимизацию не нужно и даже нежелательно.
Решать проблемы следует по мере их возникновения.
Индексы
По умолчанию база данных запоминает недавние запросы в целях увеличения быстродействия.
Создание индекса
ПКМ по интересующей таблице -> ALTER TABLE -> Вкладка "Indexes" -> Задаём имя в Index name,
Type = Index -> Выбираем столбец, для которого применить индекс -> Apply
Запрос на языке SQL:
	ALTER TABLE `shop`.`product` 
	ADD INDEX `price_index` (`price` ASC) VISIBLE;
	;
