Docker Swarm - это механизм организации контейнеров, встроенный непосредственно в Docker.
Это всего лишь группа машин (механизмов Docker), выполняющих контейнеры в режиме Swarm.

Сущности Docker Swarm:
  * Управляющие узлы
    = "мозг" кластера.
      Отвечают за управление рабочими узлами, где выполняются контейнеры.
      В кластере Swarm имеется несколько управляющих узлов (обычно нечётное количество, что помогает избежать
      проблемы с единой точкой отказа).
      У.У. выполняют алгоритм Raft-консенсуса для выбора единственного лидера.
    #(Если по какой-то причине лидер остановится, на его роль будет выбран один из оставшихся управляющих узлов
      => удастся исключить возможность простоя из-за сбоев).
